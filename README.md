# Personalized News Aggregator

## Project Description
This project is a personalized news aggregator that scrapes articles from multiple sources, categorizes them using natural language processing (NLP), and provides access through a REST API and a front-end interface.

## Features
- Scrapes news articles from various sources.
- Categorizes content using NLP libraries like NLTK, spaCy, or TextBlob.
- REST API for accessing articles.
- Front-end interface built using React.
